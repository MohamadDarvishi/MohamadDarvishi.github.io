Questions and answers about Arad font

What is Arad font license?

Arad font, like other open source fonts, is licensed by OFL. Anyone can use it for any purpose.

What languages ​​does Arad font support?

Arad font supports Farsi, Arabic, Türkcə, Urdu, Kurdi, Mazeruni and English languages.

What is the difference between Arad font and Shabnam font?

Although the Shabnam font was still in use, it was an archived project and its bugs were not fixed and it was not as developed as Vazirmatn Font. Arad font is made based on the structure and purpose of Shabnam font, so that it is bigger and does not have the defects of Shabnam. Arad font has been redesigned and has a better design, more weight, new modes, support for more languages ​​and...

In what category is the Arad font?

Arad font is of Sans type, but it is placed in the category of geometric fonts in Persian fonts.

How many weight/dots style does Arad font have?

8 weights and three dot styles. A total of 24 modes

Weight:
  • Thin
  • ExtraLight
  • Light
  • Regular
  • Medium
  • SemiBold
  • Bold
  • ExtraBold
    • Dots Style:
  • 1- Discrete square dots
  • 2- Continuous square dots

  • 4- Circle dots

    How to show English numbers as Persian numbers?

    This issue can be changed in different ways and English numbers can be displayed in the form of Persian numbers.

    First solution: Just enable the ss01 feature in the app, which is the best solution. With this solution, you can have Persian numbers in some places and English numbers in some places without using multiple fonts.

    If you use fonts on the web, you can use this code sample in the site code.

    .sample_farsi_digits {
  font-feature-settings: "ss01";
}

    The second solution: use the Persian number font. These artificially or hacked fonts show Farsi numbers instead of English numbers, which is not recommended, but you can do it.

    What is the font of English letters of Arad font?

    The font is Kanit. Of course, this font has had changes, and it has been changed into a variable so that it can be used in Arad font.

    Github: https://github.com/cadsondemak/kanit
