نحوۀ استفاده از فونت سورنا در آرت های پیکسلی

فونت سورنا را فارغ از هر نرم‌افزاری برای طراحی، می‌توان استفاده کرد.

اندازهٔ سورنا Pixel باید ضریبی از ۶.۴ و اندازهٔ سورنا Normal ضریبی از ۲۵.۶ باشد تا فونت به طور صحیح در آرت پیکسلی مورد استفاده قرار گیرد.

نکتهٔ قابل توجه این است که سورنا چون یک فونت پیکسلی است نیاز به حالت smooth یا رندر های anti-aliasing نیاز ندارد و فعال بودن این ویژگی ها سبب رندر اشتباه فونت سورنا می‌شود.

پرسش و پاسخ

فونت سورنا تحت چه مجوزی است؟

فونت سورنا همچون فونت های دیگر تحت مجوز OFL است. هر کسی می‌تواند از آن برای هر قصدی استفاده کند.

فونت سورنا از چه زبان هایی پشتیبانی می‌کند؟

فونت سورنا از زبان های فارسی، عربی، اردو، کوردی و انگلیسی پشتیبانی می‌کند.

فونت سورنا جزو چه دسته‌ای از فونت هاست؟

فونت سورنا جزو دستۀ پیکسل‌فونت هاست.

فونت سورنا چه حالت هایی دارد؟

فونت سورنا دو حالت دارد. حالت عادی(Normal) و حالت قاعده‌مند(Pixel). حالت قاعده‌مند از مربع هایی با اندازه های یکسان و در فاصله‌های یکسان ساخته شده؛ در صورتی که بخاطر زیبایی در حالت عادی چنین چیزی وجود ندارد.

چگونه می‌شود مشکلات فونت را گزارش داد؟

مشکلات فونت را می‌توان از طریق صفحۀ مسائل فونت گزارش داد

چگونه می‌توان فونت را روی ویندوز نصب کرد؟

کافی‌ست بعد دانلود فایل زیپ، و استخراج آن، فایل ttf فونت را باز کرده و دکمۀ Install از قسمت بالا چپ را کلیک کنید.

چرا هنگام نوشتن اعداد، اعداد انگلیسی نمایش داده می‌شوند؟

این استاندارد است که هنگام نوشتن اعداد انگلیسی، چیزی که به شما نمایش داده شود عدد انگلیسی باشد. یا می‌توانید عدد فارسی را بنویسید یا از روش بَد زیر استفاده کنید! روش بَد: ویژگی ss01 را فعال کنید.

چگونه از روش بد در وب استفاده کنیم؟

در Css برای متن مورد نظر، کد زیر را وارد کنید 

.sample_farsi_digits {
        font-feature-settings: "ss01";
      }

نسخۀ اعداد فارسی

misc\Farsi-Digits در این پوشه، فونت های هک شده ای وجود دارند که به جای اعداد انگلیسی و عربی، اعداد فارسی می‌نویسند.
سوالم رو اینجا پیدا نکردم. باید چیکار کنم؟ در صفحۀ مسائل فونت سوال خود را بپرسید. اگر سوال مناسب باشد در اینجا هم اضافه خواهد شد.
